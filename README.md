# Delibera APP

Repositório da versão mobile do Delibera

**Em que estado se encontra**

Este projeto encotra-se em desenvolvimento e conta com uma build lançada na play store, mas ainda sem funcionalidades acessíveis para usuários

**Como colaborar**

Verifique nas issues existentes se tem algo que possa finalizar ou se há alguma funcionalidade que não existe no App e inicie uma issue.

**Como instalar**
```
git clone
npm install
npm install -g bower
bower install
ionic serve
```
